A simple web-based dictionary powered by a JSON dataset. Users can search for words and see their definitions, phonetics, origin, and examples. Audio pronunciation is supported when available.

This project is fully hosted on GitHub Pages, making it easy to access online.

⸻

Features
	•	Search for words from a custom dictionary dataset.
	•	Displays:
	•	Word
	•	Phonetic transcription
	•	Audio pronunciation
	•	Part of speech
	•	Definitions
	•	Example sentences
	•	Simple and clean user interface.
	•	Works entirely in the browser (no backend needed).

How to Use
	1.	Type a word into the search input.
